## Описание программы
Book Tracker - это桌面 приложение для управления списком прочитанных книг. Программа позволяет:
- Добавлять книги с указанием названия, автора, жанра и количества страниц
- Фильтровать книги по жанру и количеству страниц
- Сохранять и загружать данные в формате JSON
- Просматривать статистику (общее количество книг, страниц, популярные жанры)
- Удалять записи двойным щелчком мыши

## Требования
- Python 3.7 или выше
- Tkinter (обычно входит в стандартную установку Python)
